## Tecnologias Back-end ⚙️
As tecnologias utilizadas foram:

- 🍃 **Java /Spring Framework** — Para construção da API

- 📶 **Postman** — Para teste da API

- 📙  **PostgreSQL 12** — Como banco de dados

- 🧰  **Heroku** — Para deploy

- 📄  **Swagger** - Para documentação da API
